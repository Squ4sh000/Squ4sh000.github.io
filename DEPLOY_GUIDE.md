# 个人主页 GitHub Pages 部署指南

你可以通过以下步骤将你的个人主页部署到 `https://Squ4sh000.github.io/`。

## 1. 创建专门的仓库
为了让你的主页显示在顶级域名 `github.io` 下，你需要创建一个名为 **`Squ4sh000.github.io`** 的公开仓库。

## 2. 上传文件
将 `personal-website` 目录下的所有内容上传到该仓库的 `main` 分支。
*   `index.html` (必须位于根目录)
*   `style.css`
*   `images/` (包含图标)

## 3. 启用服务
通常对于以 `.github.io` 结尾的仓库，上传文件后 GitHub Pages 会自动启用。
1.  进入仓库 **Settings** > **Pages**。
2.  确认 **Build and deployment** > **Branch** 设置为 `main`。
3.  等待几分钟后，访问 `https://Squ4sh000.github.io/` 即可看到你的精美主页。

## 4. 网站功能说明
*   **鼠标追踪光效**：背景会随鼠标移动产生微弱的青色霓虹光晕。
*   **响应式设计**：完美适配手机、平板和电脑屏幕。
*   **项目集成**：主页已重点展示了 **N Language**，并提供了源码和官网链接。
*   **技能展示**：清晰罗列了你在底层开发和工具生态方面的专长。
